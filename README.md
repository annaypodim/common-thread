# common-thread
A web platform that helps students strategically build applications. Includes an application angle analyzer, essay feedback that preserves student voice, and a Notion-style dashboard for organizing college research, essays, deadlines, and application materials.

## Supabase profile tables
Run `supabase/profile_schema.sql` in your Supabase SQL editor to enable persistent profile storage for:
- high school + intended majors
- activities/extracurriculars
- honors/awards
